# FRR_Lab
Vagrant FRR Lab with OSPF redistribution into BGP

## Net diagram
![FRR_Lab](vagrant_frr_lab_ospf2bgp.jpeg)

## Requisites
- Vagrant
- Virtualbox
- Ansible

## Basic usage
- Clone this repo
- vagrant up


## Connect to instances
- vagrant ssh core
- vagrant ssh asbr0
- vagrant ssh pe0
- etc
